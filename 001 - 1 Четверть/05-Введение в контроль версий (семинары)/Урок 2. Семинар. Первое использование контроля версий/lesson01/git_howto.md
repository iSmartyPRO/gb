# Подсказка по GIT

## Создание репозитория
```sh
git init
```

## Добавление файла в индекс (staging area) для последующего коммита
```sh
git add <filename>
```

## Отображение состояния файлов в рабочем каталоге и индексе
```sh
git status
```
