## Задача 1: 
```
Напишите программу, которая бесконечно
запрашивает целые числа с консоли. Программа
завершается при вводе символа ‘q’ или при вводе
числа, сумма цифр которого четная.

Пример:
5 12 16 q [STOP]
3 45 342 15 [STOP]
```
