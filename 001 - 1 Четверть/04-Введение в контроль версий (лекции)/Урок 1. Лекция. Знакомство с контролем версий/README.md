# Урок 1. Лекция. Знакомство с контролем версий

### Получение версии Git
```sh
git --version 
git version 2.42.0.windows.1
```

### Инициализация Git
```sh
git init
```

### Получение статуса
```sh
git status
```

### Добавление файлов
> можно указать названия файлов либо точкой указав все файлы.
```sh
git add .
```
